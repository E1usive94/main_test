# Новый Вектор 2026

Сайт проекта «Новый Вектор 2026».

## Структура

- `index.html` — главная
- `about.html` — что такое «Новый Вектор 2026»
- `services.html` — службы
- `services/*.html` — отдельные страницы служб
- `team.html` — наша команда
- `schedule.html` — план мероприятий

## Публикация на GitHub Pages

1. Создайте репозиторий на GitHub.
2. Загрузите все файлы и папки из архива в корень репозитория.
3. Откройте `Settings` → `Pages`.
4. Выберите:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
5. Нажмите `Save`.
