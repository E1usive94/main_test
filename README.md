# Новый Вектор 2026 — многостраничный сайт

Тестовая версия сайта для GitHub Pages.

## Страницы

- `index.html` — главная
- `about.html` — что такое «Новый Вектор 2026»
- `services.html` — раздел со службами
- `services/*.html` — отдельные страницы служб
- `team.html` — наша команда
- `schedule.html` — план мероприятий

## Как загрузить на GitHub Pages

1. Создайте новый репозиторий на GitHub.
2. Загрузите все файлы и папки из этого архива в корень репозитория.
3. Откройте `Settings` → `Pages`.
4. В блоке `Build and deployment` выберите:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
5. Нажмите `Save`.
6. Через 1–2 минуты GitHub покажет публичную ссылку на сайт.
