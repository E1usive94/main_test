# Новый Вектор — 2026

Многостраничный статический сайт для публикации на GitHub Pages.

## Страницы

- `index.html` — главная
- `services.html` — каталог служб
- `services/*.html` — отдельные страницы служб
- `about.html` — о проекте
- `contacts.html` — контакты

## Как опубликовать на GitHub Pages

1. Создай новый репозиторий на GitHub, например `new-vector-2026`.
2. Загрузи все файлы из этой папки в репозиторий.
3. Открой `Settings` → `Pages`.
4. В разделе `Build and deployment` выбери:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
5. Нажми `Save`.
6. Через пару минут сайт будет доступен по адресу вида:
   `https://USERNAME.github.io/new-vector-2026/`
